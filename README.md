# HR-Analytics-Dashboard-Excel
This project is an interactive HR Analytics Dashboard built using Microsoft Excel. The goal of this project is to analyze employee attrition and identify patterns across departments and job roles.

## Key Features
- Data cleaning using Power Query
- Data analyis using Pivot Table
- Interactive dashboard using charts and slicers
- Attrition analysis by Department, Job Role, and Gender
- Average salary comparison across depatments

## Insights
- Identified departments with higher employee attrition
- Analyzed job roles with maximum employee turnover
- Compared average salary across departments to understand possible attrition factors

## Conclusion
This Dashboard helps in understanding employee attrition trends and supports data-driven decision making for HR teams.
